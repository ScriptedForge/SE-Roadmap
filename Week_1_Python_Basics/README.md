# Fitness Check-In Program

A Python-based fitness tracking program that allows users to compare their current weight against their starting weight and goal weight.

The program calculates:
- Current BMI
- Weight progress
- Goal status
- BMI category

## Features

- User profile tracking
- Weight goal comparison
- BMI calculator
- Progress feedback using conditional logic
- User-friendly terminal output

## Concepts Practiced

This project was created while learning Python fundamentals.

Topics used:
- Variables
- User input
- Data types
- Type conversion
- Arithmetic calculations
- f-strings
- if / elif / else statements
