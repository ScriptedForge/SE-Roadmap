# Week 3 - Modules & Data Structures

## Overview

This week focused on organizing larger Python programs by learning when and how to use different data structures, as well as separating code into reusable modules. I learned how Python manages namespaces, imports, and shared objects across multiple files while continuing to improve my ability to reason through code before executing it.

## Concepts Covered --------------------------------------------------------------------

### Data Structures
- Tuples
- Sets
- Dictionaries
- Choosing the appropriate data structure for different situations

### Modules & Imports
- Creating custom modules
- Importing modules and individual objects
- Import aliases
- Namespaces
- Module state
- Shared mutable objects
- Organizing projects across multiple files

## Projects & Exercises

- Refactored an RPG project into multiple Python modules
- Separated character data, combat logic, and display functions into dedicated files
- Used imports to coordinate functionality between modules
- Practiced predicting program behavior before execution

## Skills Developed
- Improved project organization
- Reduced code duplication through modular design
- Learned to structure programs for scalability
- Strengthened debugging by mentally tracing execution flow
- Gained confidence working with larger multi-file projects
